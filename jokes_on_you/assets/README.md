# Assets

## `uncle_sam.png`

Drop the classic James Montgomery Flagg **"I Want YOU for U.S. Army"** poster
here as `uncle_sam.png`. It is public domain (pre-1929 U.S. government /
Library of Congress).

### Where to get it

* Library of Congress: https://www.loc.gov/pictures/item/2010715864/  (download the TIFF, convert to PNG)
* Wikimedia Commons: https://commons.wikimedia.org/wiki/File:I_Want_You_for_the_U.S._Army.jpg

After downloading, place the file at:

    jokes_on_you/assets/uncle_sam.png

If the file is missing, the program falls back to a procedurally drawn
red/white/blue "I WANT YOU" poster — less iconic but functional.
